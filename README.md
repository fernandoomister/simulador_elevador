#  Simulador de Elevador

##  Descrição
Este projeto consiste em um simulador de elevador desenvolvido em Portugol Studio.  
O sistema permite ao usuário interagir com o elevador, escolhendo andares, visualizando a posição atual e simulando o funcionamento real de um elevador.

---

##  Objetivo
Desenvolver um sistema interativo que simule o comportamento de um elevador, aplicando conceitos de lógica de programação como:
- Estruturas de decisão (se/senao)
- Estruturas de repetição (enquanto)
- Variáveis e controle de fluxo

---

##  Funcionalidades

✔ Escolher andar (0 a 6)  
✔ Mostrar andar atual  
✔ Sistema interativo com menu  
✔ Controle de capacidade (máximo 5 pessoas)  
✔ Validação de entradas  
✔ Simulação de subida e descida  
✔ Feedback em tempo real  
✔ Simulação de som (Ding!)

---

##  Lógica do Sistema

O sistema funciona com base em um menu interativo que permite ao usuário selecionar opções.  
O elevador movimenta-se andar por andar, exibindo o estado atual durante o percurso.

---

##  Como Executar

1. Abrir o Portugol Studio  
2. Carregar o arquivo do projeto (.port)  
3. Executar o programa  
4. Interagir com o menu apresentado
